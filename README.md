# Dharshan India Traders (DIT)

Corporate website for **DHARSHAN INDIA TRADERS PRIVATE LIMITED** — construction materials trading company based in Avudaiyarkoil, Pudukkottai, Tamil Nadu.

## Quick start

Open `index.html` in a browser, or serve the folder locally:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```

Then visit `http://localhost:8080`.
